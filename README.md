# euishoeProject
하나금융티아이 교육 shoppingmall 프로젝트

## Branch Naming Rule  
-feature  
> - 기능명(도메인)_닉네임  
> - ex)login(customer)_sw  
- realease tag  
> - 1(1로 고정).0(release기준점마다+1).0(hotfix생길때마다+1)  
> - ex)1.1.3(첫번째 release버전에서 hotfix가 3가지가 생겼고 모두 완료한경우)  
- hotfix  
> - 응급수정사항  
> - master에서 따온 branch에서 바로 수정  
> - 수정 후 develop, master와 모두 merge  
> - 응급수정기능명_닉네임  
> - ex)login_sw
